          Preferences and data

          Locations and preferences are stored only in this browser. When weather is requested, the selected coordinates are sent to the named weather providers. There is no account and no server-side profile.

          Sources

          • Weather forecast by Open-Meteo, using automatically selected high-resolution models.
          • Two-hour rain data by Buienradar.nl.
          • Belgian observations by KMI/IRM.
          • Location search data by GeoNames through Open-Meteo.

          Weather can change quickly. Use official warnings for safety-critical decisions.
