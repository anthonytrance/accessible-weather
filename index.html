          Notifications

          Get a push notification shortly before rain starts, and an optional daily briefing. Alerts follow one chosen place.

          Preferences and data

          Locations and preferences are stored only in this browser. When weather is requested, the selected coordinates are sent to the named weather providers. If notifications are enabled, the chosen alert location and language are stored on the notification server. There is no account and no profile.

          Sources
          • Open-Meteo: weather forecast using automatically selected high-resolution models.
          • Buienradar.nl: two-hour rain nowcast.
          • KMI/IRM: Belgian station observations.
          • Open-Meteo Air Quality: air quality and pollen (CAMS).
          • Location search data by GeoNames through Open-Meteo.

          Weather can change quickly. Use official warnings for safety-critical decisions.
